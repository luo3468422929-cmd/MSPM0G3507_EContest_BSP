# 默认引脚资源分配表

默认分配依据用户提供的预期引脚图，并由 SysConfig 1.26.2 对 MSPM0G3507 LQFP-64 完成资源验证。

| 模块 | 信号 | 默认引脚/外设 | 说明 |
|---|---|---|---|
| TB6612 | PWMA | PA12 / TIMG0_C0 | 左电机 PWM |
| TB6612 | PWMB | PA13 / TIMG0_C1 | 右电机 PWM |
| TB6612 | AIN1/AIN2 | PA8 / PA9 | 左电机方向 |
| TB6612 | BIN1/BIN2 | PB18 / PA7 | 右电机方向 |
| TB6612 | STBY | PB24 | 高电平使能 |
| OLED | SCL/SDA | PB2 / PB3 / I2C1 | 默认硬件 I2C |
| 调试串口 | TX/RX | PA28 / PA31 / UART0 | 115200 8N1 |
| 串口惯导 | TX/RX | PA26 / PA25 / UART3 | 默认 115200 8N1 |
| 左编码器 | A/B | PA21 / PA22 | A 相双边沿中断，B 相判向 |
| 右编码器 | A/B | PB19 / PB20 | A 相双边沿中断，B 相判向 |
| 板载 LED | LED | PA14 | 默认高电平点亮 |
| 按键 | KEY | PB6 | 默认低电平按下，内部上拉 |
| 寻迹 0～7 | D0～D7 | PA24、PB9、PB8、PA18、PA17、PA16、PA15、PA23 | 默认数字模式 |

## 重要说明

1. PA23 在原图标注过 ADC 参考相关用途，默认数字模式使用前必须根据实际板卡焊接与 VREF 方案确认；若使用外部 ADC 参考，请重新分配 D7。
2. PA27 在 `.syscfg` 中仅保留为输入占位，不参与当前业务，可在比赛时释放。
3. 模拟八路灰度模式必须重新选择 8 个 ADC 可用引脚，不应机械照搬数字寻迹引脚。
4. 切换 OLED SPI 时，需要在 SysConfig 增加 SPI、CS、DC、RESET，并同步 `bsp_config.h`。
5. 修改引脚后应重新运行 SysConfig 验证，不能只修改 C 宏。
