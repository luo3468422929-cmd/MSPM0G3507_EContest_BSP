# 地猛星 MSPM0G3507 当前引脚表

以 `empty.syscfg` 为唯一真源，器件为 MSPM0G3507 LQFP-48(PT)。

| 模块 | 信号 | MCU 引脚 / 外设 |
|---|---|---|
| SWD | SWDIO / SWCLK | PA19 / PA20 |
| 板载 LED | LED | PA14 |
| 板载 CH340 | UART0 TX / RX | PA10 / PA11，115200 |
| TB6612 PWM | PWMA / PWMB | PA12 / PA13，TIMG0 C0/C1 |
| TB6612 左方向 | AIN1 / AIN2 | PB18 / PA7 |
| TB6612 右方向 | BIN1 / BIN2 | PA8 / PA9 |
| TB6612 | STBY | PB24，默认低 |
| 左编码器 | A / B | PA17 / PA16，GPIO 上升沿中断 |
| 右编码器 | A / B | PB19 / PB20，GPIO 上升沿中断 |
| NCU 惯导 | UART2 TX / RX | PA21 / PA22，115200 |
| 八路循迹 | I2C0 SDA / SCL | PA28 / PA31，400 kHz |
| ST7735S SPI | SCK / MOSI | PB9 / PB8，SPI1 4 MHz |
| ST7735S 控制 | CS / DC / RES | PA27 / PA26 / PA25 |
| ST7735S 背光 | BL | 直连 3.3 V，不占 MCU |
| 临时按键 | KEY | PB2，内部上拉，按下接 GND |

保留/空闲约束：PA0、PA1 当前不配置且不作为首选 I2C；PA2 为板载 ROSC 相关保留；PA24 因 BL 硬接 3.3 V 而留空。

电机或编码器方向不一致时，只改 `User/user_config.h` 中对应的 `*_REVERSED`。
