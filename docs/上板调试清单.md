# 上板调试清单

测试选择统一修改 `User/user_config.h` 的 `STARTUP_TEST`。上一项不通过，不进入下一项。

1. `TEST_LCD`：确认四色块、`codex nb`、中文“电赛”和棋盘；BL 直连 3.3 V。
2. `TEST_KEY`：PB2 到 GND；确认 UART0/CH340 能看到按键事件。
3. `TEST_IMU`：模块 TX→PA22、RX←PA21、共地；确认 yaw/gyroZ 和 ONLINE。
4. `TEST_TRACK`：5V/GND、SDA=PA28、SCL=PA31；K1/K2 标定后逐路检查 001~800，静止时 `i2c=OK`。
5. `TEST_ENCODER`：手转检查正负方向、静止零速和十圈约 4500 计数。
6. `TEST_MOTOR`：架空，确认 VM/VCC/共地/STBY；正转 5 s、停 1 s、反转 5 s；按 PB2 急停。
7. `TEST_PID`：保持架空，确认左右 40 RPM 闭环和 VOFA 曲线；按 PB2 急停。
8. 恢复 `TEST_NONE`：上电应先 ARMING，循迹就绪后自动 RUN；按 PB2 锁定 STOPPED。
9. 低速落地，再逐步提高 `CAR_DEFAULT_BASE_SPEED_RPM`。
10. 运行 `Tests/Build/verify_build.ps1` 并提交 Git。
