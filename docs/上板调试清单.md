# 上板调试清单

严格按顺序执行，上一项不通过不要进入下一项。

1. **LCD**：`Test_Select(TEST_LCD)`，确认红绿蓝白四色、`LCD OK`、中文“电赛”和右下角棋盘图片。整体偏移只调 `LCD_X/Y_OFFSET`；颜色/方向再调 `LCD_MADCTL`。
2. **八路 I2C 寻迹**：`TEST_TRACK`，模块 5V/GND 接电源，SDA 接 PA28、SCL 接 PA31。依次让黑线经过 X1~X8，串口位图应依次为 01/02/04/08/10/20/40/80；地址为 0x12、寄存器为 0x30，先完成模块黑白标定。
3. **按键**：`TEST_KEY`，短按应看到 PRESSED/CLICKED，长按应看到 LONG；接线为 PB2 到 GND。
4. **电机（车轮架空）**：先断开负载，确认 TB6612 STBY 和电源；`TEST_MOTOR` 只在明确选择后正反转一次。方向错只改 `MOTOR_*_REVERSED`。
5. **编码器方向**：`TEST_ENCODER`，手转左/右轮，计数应连续且前进方向为正；错误只改 `ENCODER_*_REVERSED`。
6. **惯导**：`TEST_IMU`，确认 UART2 收到连续 yaw/gyroZ，状态 ONLINE；模块 TX 接 PA22，PA21 到模块 RX 串 1 kΩ。
7. 把 `Test_Select(...)` 改回 `TEST_NONE`，落地低速测试短按启停、长按急停和丢线停车。

完成后运行 `Tests/Build/verify_build.ps1` 并提交 Git。
