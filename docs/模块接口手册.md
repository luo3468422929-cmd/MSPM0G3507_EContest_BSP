# 模块接口手册

公开函数统一采用 `Module_Action()`，不添加冗余的 `BSP_` 前缀。`User` 不直接调用 TI DriverLib。

## Bsp

- `Board_Init()`：调用 SysConfig 并按依赖顺序初始化全部启用模块。
- `Board_EmergencyStop()`：立即关闭 TB6612。
- `Timer_GetTickMs()`、`Timer_ElapsedMs()`、`Timer_DelayMs()`：1 ms 时基。
- `UART_Send()`、`UART_ReadByte()`、`UART_Printf()`：调试口/惯导口统一收发；RX 中断只写环形缓冲。
- `Command_Register()`、`Command_Process()`：不定长 AA55 指令帧解析。
- `SPI_Write()`：带超时的 SPI1 阻塞发送，只由硬件驱动调用。
- `I2C_ReadRegister()`、`I2C_WriteRegister()`：带有限超时的 I2C0 寄存器读写。

## Hardware

- `Motor_Init()`、`Motor_Enable()`、`Motor_SetDutyPair()`、`Motor_RampUpdate()`、`Motor_EmergencyStop()`：TB6612 双电机。
- `Encoder_Init()`、`Encoder_UpdateSpeed()`、`Encoder_GetData()`：两路捕获测速。
- `Track_Init()`、`Track_Update()`、`Track_GetData()`：亚博八路 MCU 灰度模块 I2C 寻迹。
- `Track_Data_t.activeMask`：bit0=X1，bit7=X8；`communicationOk` 表示最近一次 I2C 读取结果。
- `Track_SetRawSamples()`：八路离线算法测试的数据注入口；未来 12 路模块应单独增加协议适配。
- `IMU_Init()`、`IMU_Process()`、`IMU_GetSample()`、`IMU_IsOnline()`：UART 惯导接收层。
- `ImuProtocol_PushByte()`：当前 0x5A 五字节协议解析器，协议更换时只替换此模块。
- `LCD_Init()`、`LCD_Fill()`、`LCD_ShowString()`、`LCD_ShowInt()`、`LCD_ShowFloat()`：ST7735S 直写显示，无全屏缓冲。
- `LCD_ShowChinese16()`：显示工程内置的 16×16 UTF-8 常用中文字，当前字库面向电赛状态页按需收录。
- `LCD_ShowBitmap()`：显示行优先、高字节在前的 RGB565 原始图片；图片必须完整落在 128×160 屏幕内。
- `Key_Scan()`、`Key_GetEvent()`：消抖、单击、长按。
- `LED_Set()`、`LED_Toggle()`：板载 LED。

## Control

- `PID_Init()`、`PID_UpdatePosition()`、`PID_UpdateIncremental()`：位置式/增量式 PID，含限幅、积分分离和死区。
- `CarControl_Init()`、`CarControl_Enable()`、`CarControl_Update()`、`CarControl_Stop()`：寻迹、转向 PID、左右轮速度 PID 和电机输出的统一协调器。
- `Scheduler_Init()`、`Scheduler_IsDue()`：5/10/20/100/200 ms 协同调度。

## User

- `System_Init()`：硬件初始化。
- `Task_Init()`：控制器和任务状态初始化。
- `Task_Run()`：比赛主状态机。
- `Test_Select()`、`Test_Run()`：单模块上板测试。

所有返回 `Status_t` 的接口都应检查 `STATUS_OK`。中断函数只做收字节、计数或置标志，不运行显示、PID、延时和格式化输出。

LCD 中文扩展不依赖商家 STM32 头文件。中文源代码使用 UTF-8；如果工程编码设置不确定，可使用 UTF-8 转义字节串。图片数组放在 `const` Flash 区，每像素两个字节，顺序为 RGB565 高字节、低字节。
